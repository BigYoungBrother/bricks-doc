gitbook使用方法：

 **运行gitbook服务**
 1. 安装node
 2. 下载gitbook包`npm install -g gitbook-cli`
 3. 在本项目根目录下执行`gitbook serve --port 23333`后在浏览器中输入`127.0.0.1:23333`后即可
