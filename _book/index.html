下载到本地的查看方法：

  1. markdown编辑器，推荐Typora和vs code + markdown all in one插件

  2. gitbook

    gitbook使用方法： 运行gitbook服务

  3. 安装node
  4. 下载gitbook包npm install -g gitbook-cli

  5. 在本项目根目录下执行gitbook serve后在浏览器中输入127.0.0.1:4000后即可

    直接查看gitbook生成的html文件 在本项目根目录下_book文件夹中有对应的html文件
